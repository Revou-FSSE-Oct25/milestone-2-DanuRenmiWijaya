# Revou Student Overview

## This is a website created to showcase my journey as a student at RevoU. This website will contain information about this project, in this project i created a simple game.

## The Features in this project are follow :
- Number Guessing Game section : Player can play and tries to guess the number
- Rock, Paper, Scissors Section : The player chooses Rock, Paper, or Scissors, and than The computer randomly selects one as well.
The game determines the winner based on classic rules.
- Clicker Game : The player click a big button appears on the screen. Every time the player clicks it, the score increases.
